## [1.0.0] - 2026-06-08

### 资料库功能增强

#### 新增功能

- **文件夹颜色标识**
  - 支持 8 种颜色（蓝色、绿色、橙色、红色、灰色、紫色、青色、天蓝）
  - 新建/编辑文件夹时可选择颜色
  - 列表视图和网格视图中显示自定义颜色

- **置顶功能**
  - 文件和文件夹支持置顶
  - 置顶项优先显示
  - 置顶图标标识

- **macOS 分栏视图**
  - 类似 Finder 的分栏文件夹浏览模式
  - 点击文件夹在右侧展开子栏
  - 支持多级嵌套导航

- **右键菜单**
  - 文件夹菜单：打开、新建子文件夹、上传文件、重命名、设置颜色、置顶、收藏、复制、移动、删除
  - 文件菜单：预览、在线编辑、下载、重命名、置顶、收藏、分配学生、复制、移动、查看详情、删除

- **拖拽移动**
  - 支持拖拽文件/文件夹到目标文件夹
  - 分栏视图支持拖拽到根目录
  - 拖拽时显示视觉反馈
  - 移动后自动刷新列表

#### 问题修复

- **OnlyOffice JWT token 冲突**
  - 修复 JwtAuthenticationFilter 解析 OnlyOffice token 时的 NumberFormatException
  - 添加异常处理，跳过非用户 token

- **OnlyOffice 回调保存失败**
  - 修复回调 URL 地址转换问题
  - 将 localhost:8081 转换为 Docker 内部地址 onlyoffice:80

- **分栏模式创建子文件夹**
  - 修复创建文件夹总是在根目录的问题
  - 现在正确创建在当前选中的文件夹下

- **拖拽移动刷新**
  - 修复移动成功后显示"移动失败"的问题
  - 修复移动后列表不自动刷新的问题

#### 数据库变更

- V15: materials 表添加 `color` 字段（文件夹颜色）
- V16: materials 表添加 `is_pinned` 字段（置顶标记）

#### API 变更

- `POST /api/v1/materials/{id}/pin` - 置顶/取消置顶
- `PUT /api/v1/materials/{id}/move` - 移动文件/文件夹

#### 前端变更

- 新增 `ContextMenu.vue` 右键菜单组件
- `MaterialListView.vue` 添加分栏视图、拖拽功能
- `MaterialForm.vue` 添加颜色选择器
- `material.ts` API 类型更新
