## 项目简介

家教助手是一款专为家教老师设计的教学管理工具，帮助老师高效管理学生信息、课程安排、教学资料、作业批改等日常工作。

### 技术架构

- **前端**: Vue 3 + TypeScript + Element Plus
- **后端**: Spring Boot 3.2 + MyBatis Plus
- **数据库**: PostgreSQL 15
- **缓存**: Redis 7
- **对象存储**: MinIO
- **文档编辑**: OnlyOffice Document Server
- **部署方式**: Docker Compose

## 系统部署

### 环境要求

- Docker 20.10+
- Docker Compose 2.0+
- 至少 4GB 可用内存

### 部署步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/beishan/aiteacher.git
   cd aiteacher
   ```

2. **配置环境变量**
   ```bash
   cp .env.example .env
   # 编辑 .env 文件，配置数据库密码等
   ```

3. **启动服务**
   ```bash
   docker-compose up -d
   ```

4. **访问系统**
   - 前端地址: http://localhost
   - 后端 API: http://localhost:8080
   - OnlyOffice: http://localhost:8081
   - MinIO 控制台: http://localhost:9001

### 服务说明

| 服务 | 端口 | 说明 |
|------|------|------|
| frontend | 80 | 前端 Web 应用 |
| backend | 8080 | 后端 API 服务 |
| postgres | 5432 | PostgreSQL 数据库 |
| redis | 6379 | Redis 缓存 |
| minio | 9000/9001 | MinIO 对象存储 |
| onlyoffice | 8081 | OnlyOffice 文档编辑 |

### 常用命令

```bash
# 启动所有服务
docker-compose up -d

# 停止所有服务
docker-compose down

# 重新构建并启动
docker-compose up --build -d

# 查看服务状态
docker-compose ps

# 查看服务日志
docker-compose logs -f [服务名]

# 重新部署单个服务
docker-compose up --build -d [服务名]
```

## 功能模块

### 资料库管理

资料库用于管理教学资料，支持文件夹分类、标签管理、在线编辑等功能。

#### 视图模式

系统提供三种视图模式：

1. **网格视图** (默认)
   - 卡片式展示
   - 文件夹和文件分开显示
   - 支持拖拽移动

2. **列表视图**
   - 表格式展示
   - 显示详细信息
   - 支持排序

3. **分栏视图**
   - macOS Finder 风格
   - 点击文件夹展开子栏
   - 适合深层目录浏览

#### 文件夹操作

| 操作 | 说明 |
|------|------|
| 新建文件夹 | 点击"新建文件夹"按钮 |
| 设置颜色 | 编辑文件夹时选择颜色（8种可选） |
| 置顶 | 右键菜单或操作栏选择"置顶" |
| 收藏 | 右键菜单或操作栏选择"收藏" |
| 重命名 | 右键菜单选择"编辑信息" |
| 删除 | 右键菜单选择"删除" |
| 移动 | 拖拽到目标文件夹 |

#### 文件操作

| 操作 | 说明 |
|------|------|
| 上传文件 | 点击"上传资料"按钮，支持拖拽上传 |
| 预览 | 支持 PDF、图片、Office 文件预览 |
| 在线编辑 | 支持 Word、Excel、PPT 在线编辑 |
| 下载 | 右键菜单选择"下载" |
| 置顶/收藏 | 右键菜单或操作栏 |
| 分配学生 | 右键菜单选择"分配给学生" |
| 移动 | 拖拽到目标文件夹 |

#### 右键菜单

在任意文件或文件夹上右键点击，可快速执行常用操作：

**文件夹右键菜单：**
- 打开
- 新建子文件夹
- 上传文件到此文件夹
- 重命名
- 设置颜色
- 置顶/取消置顶
- 收藏/取消收藏
- 复制
- 移动到...
- 删除

**文件右键菜单：**
- 预览
- 在线编辑
- 下载
- 重命名
- 置顶/取消置顶
- 收藏/取消收藏
- 分配给学生
- 复制
- 移动到...
- 查看详情
- 删除

#### 拖拽移动

1. 按住文件或文件夹
2. 拖拽到目标文件夹
3. 松开鼠标完成移动
4. 在分栏视图中，可拖拽到第一列移回根目录

#### 搜索功能

- 支持按标题搜索
- 支持按标签搜索
- 支持按科目、年级筛选

## 常见问题

### Q: 如何重新部署服务？

```bash
# 重新部署所有服务
docker-compose up --build -d

# 重新部署单个服务
docker-compose up --build -d backend
docker-compose up --build -d frontend
```

### Q: 如何查看服务日志？

```bash
# 查看所有服务日志
docker-compose logs -f

# 查看指定服务日志
docker-compose logs -f backend
docker-compose logs -f frontend
```

### Q: 如何备份数据库？

```bash
# 备份 PostgreSQL
docker exec tutor-postgres pg_dump -U postgres tutor_assist > backup.sql

# 恢复 PostgreSQL
cat backup.sql | docker exec -i tutor-postgres psql -U postgres tutor_assist
```

### Q: OnlyOffice 编辑器无法使用？

1. 检查 OnlyOffice 容器是否运行：`docker ps | grep onlyoffice`
2. 检查日志：`docker logs tutor-onlyoffice`
3. 确保 JWT_SECRET 配置一致

### Q: 文件上传失败？

1. 检查 MinIO 容器是否运行：`docker ps | grep minio`
2. 检查文件大小是否超过限制
3. 检查磁盘空间是否充足

### Q: 如何修改端口？

编辑 `docker-compose.yml` 文件，修改对应服务的端口映射：

```yaml
ports:
  - "新端口:原端口"
```

### Q: 如何配置 HTTPS？

1. 准备 SSL 证书
2. 修改 Nginx 配置
3. 配置反向代理
